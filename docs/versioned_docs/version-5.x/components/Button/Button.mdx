A button is component that the user can press to trigger an action.




  <ScreenshotTabs screenshotData={{"text":"screenshots/button-1.png","outlined":"screenshots/button-2.png","contained":"screenshots/button-3.png","elevated":"screenshots/button-4.png","contained-tonal":"screenshots/button-5.png"}} baseUrl="/react-native-paper/"/>
  

## Usage
```js
import * as React from 'react';
import { Button } from 'react-native-paper';

const MyComponent = () => (
  <Button icon="camera" mode="contained" onPress={() => console.log('Pressed')}>
    Press me
  </Button>
);

export default MyComponent;
```
